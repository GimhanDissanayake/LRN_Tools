# Description

This is a sample node app created using:
1. Node
2. Express
3. MySQL driver

To demonstrate how SQL queries are used in programming

Steps:

1. Change directory
```
cd LRN_Tools/mysql/node-mysql-app/
```

2. Install node modules
```
npm install
```

3. Start Express server
```
node index.html
```

4. Browse to
```
http://localhost:50000/users
```
